# Automatic wooden spoon
Arduino Nano based Project of an automatic wooden spoon controlled by a mobile app. (European Project - AI4VET)

### Material

#	Arduino 
*	PLA Filament 
*	3D Printer
*	DC Motor
*	Bluetooth HC-05
* Motor Driver - Dual TB6612FN
* Jumpers
*	Breadboard
*	Computer
*	Internet
*	Smartphone
